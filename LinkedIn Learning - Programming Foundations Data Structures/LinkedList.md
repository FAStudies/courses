# Linked List Operations

| Operation                 | Time Complexity |
| ------------------------- | --------------- |
| Access                    | O(n)            |
| Updating                  | O(n)            |
| Inserting at beginning    | O(1)            |
| Inserting at back, middle | O(n)            |
| Searching                 | O(n)            |
| Deleting                  | O(n)            |

**MergeSort** is preferred for sorting a LinkedList.

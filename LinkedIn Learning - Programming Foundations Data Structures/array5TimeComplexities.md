# Array: Time Complexities

| Operation          | Time Complexity |
| ------------------ | --------------- |
| Insertion          | O(n)            |
| Access             | O(1)            |
| Searching Unsorted | O(n)            |
| Deleting           | O(n)            |

- Merge Sort
